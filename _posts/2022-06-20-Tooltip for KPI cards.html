Microsoft Power BI team showcased a lot of soon to be released cool features during Microsoft Business Application Summit - May 2021. One feature which excited me the most was "Connected PivotTable refresh in Excel for the web".

This is a game changer for Power BI adoption or data adoption in general, no matter how fancy the tool is, an analyst wants the data to be in excel, why? It gives them the freedom to do further analysis.

Excel is a familiar tool for many, so many business users use excel for data analysis.

What if we have the scalability, performance of Power BI dataset connected to Excel? this is what Microsoft has created.

Why is this exciting?

As Chris Webb mentioned in his blog, this gives users three ways to present the data connected to Power BI dataset, Power BI, Paginated reports and now Excel.

Do read the blog to know more about the feature and how this will be useful

Golden datasets can be created in Power BI, once the dataset is certified and promoted in the service, users can discover these datasets from Power BI service and connect to their data analysis tool of choice

How to use?

Ruth from Crubal YouTube channel shows how to use this feature Live connect Excel and Power BI !! - YouTube

First, we have to create the connection between excel and Power BI, this can be done in below two ways

To create an Excel to Power BI dataset connection

  • Open any report or dataset from the Power BI service -> Analyze in Excel
  • Or within Excel Insert-> PivotTable-> From Power BI or Data-> Get Data-> From Power BI

To refresh and share the Excel with others

  • Drag and drop of fileds updates the data
  • The Excel can be shared through SharePoint or from Power BI workspace/ app like any Power BI report

Read more about the release here Power BI May 2021 Feature Summary | Microsoft Power BI Blog | Microsoft Power BI

When can I use this?

Microsoft announced on July 19th 2021 that the feature is made available in all tenants

If you are not able to find the feature, reach out to your tenant admin

Connected PivotTable refresh in Excel for the web is now 100% available to Office tenants | Microsoft Power BI Blog | Microsoft Power BI

Future

Future looks bright for Office 365 users, which is a major portion of business tools users. Microsoft recently (June 2021) published the "2021 release wave 2".

In which they announced Power BI integration with PowerPoint

Power BI integration with PowerPoint | Microsoft Docs

As of now, if a Power BI user wants to create PowerPoint presentation, they either have to export the report as PPT or copy selective visuals to their own PPT.

Either way, if the data changes in future, they have to redo all the steps to have an updated deck.

Imagine if the charts in the PPT gets updated with the latest data? This will be awesome right? This is what we will have in future, Office tools like PowerPoint, Excel connected live to Power BI datasets

Power BI dataset integration with office tools will help in creating a data culture without data silos.
